--HOW TO USE--
1. Setup origin Latitude, Longitude, Altitude (e.g. from Google Maps)
2. Setup target Latitude, Longitude, Altitude OR X,Y,Z in Unreal coordinates
3. Press according button

Additionaly you can move camera to converted geo
